# GitHub Profile Achievements

GitHub 프로필에 표시할 수 있는 성취(배지) 목록과 획득 조건입니다. 이 배지들은 GitHub 시스템에 의해 자동으로 부여되며, 조건 달성 후 프로필에 나타나기까지 최대 24~72시간이 걸릴 수 있습니다.

## 🏆 현재 획득 가능한 배지 (Tiers 적용)

아래 배지들은 조건 달성 횟수에 따라 등급(기본, 브론즈, 실버, 골드)이 올라갑니다.

### 1. Pull Shark
- **설명:** Pull Request(PR)를 병합(Merge)하면 획득
- **조건 (추정치):**
  - **기본:** PR 2개 병합
  - **브론즈:** PR 16개 병합
  - **실버:** PR 128개 병합
  - **골드:** PR 1,024개 병합

### 2. Starstruck
- **설명:** 소유한 리포지토리가 일정 수 이상의 Star를 받으면 획득
- **조건 (추정치):**
  - **기본:** 16 Stars
  - **브론즈:** 128 Stars
  - **실버:** 512 Stars
  - **골드:** 4,096 Stars

### 3. Pair Extraordinaire
- **설명:** 다른 사람과 공동으로 작성한 PR이 병합되면 획득 (Commit에 `Co-authored-by:` 태그 포함 필요)
- **조건 (추정치):**
  - **기본:** 공동 작성 PR 1개 병합
  - **브론즈:** 공동 작성 PR 10개 병합
  - **실버:** 공동 작성 PR 24개 병합
  - **골드:** 공동 작성 PR 48개 병합

### 4. Galaxy Brain
- **설명:** GitHub Discussions에서 작성한 답변이 '채택된 답변(Accepted Answer)'으로 선정되면 획득
- **조건 (추정치):**
  - **기본:** 답변 2개 채택
  - **브론즈:** 답변 8개 채택
  - **실버:** 답변 16개 채택
  - **골드:** 답변 32개 채택

## 🎯 단일 획득 배지 (Tiers 없음)

아래 배지들은 특정 조건을 1회만 달성하면 획득하며, 등급업이 없습니다.

### 5. Quickdraw
- **설명:** Issue나 PR이 생성된 지 5분 이내에 닫히면(Close) 획득.

### 6. YOLO
- **설명:** 코드 리뷰(Code Review) 없이 PR을 병합하면 획득 (단, Public 리포지토리여야 함).

### 7. Public Sponsor
- **설명:** GitHub Sponsors 프로그램을 통해 오픈소스 기여자나 프로젝트를 재정적으로 후원하면 획득.

## 💡 유의 사항

1. **비공개 리포지토리(Private Repo):** 기본적으로 비공개 리포지토리 활동은 배지 획득에 반영되지 않습니다. 반영하려면 프로필 설정에서 **"Show private contributions & achievements"** 옵션을 켜야 합니다.
2. **Fork 제약:** Fork한 리포지토리 내에서의 단순 활동은 종종 카운트되지 않으며, 원본 리포지토리로 PR을 보내 병합되어야 인정되는 경우가 많습니다.
3. **지연 시간:** 요건을 충족하더라도 시스템 처리 때문에 바로 배지가 보이지 않을 수 있습니다. 며칠 기다려 보는 것이 좋습니다.
4. **가시성:** 프로필 설정에서 배지 섹션 전체를 숨기거나 표시할 수 있습니다.
