## Typography direction
- **Tobias**: 300, 24px, 32px, 72px, 1.10, 1.30; substitute `Playfair Display`.
- **Atkinson Hyperlegible Mono**: 300, 10px, 12px, 14px, 16px, 1.57, 1.60, 1.63, 1.83; substitute `IBM Plex Mono`.
- **Work Sans**: 400, 12px, 16px, 1.63, 1.83; substitute `Inter`.

## Spacing / shape
- Section Gap: `64px`
- Card Padding: `24px`
- Element Gap: `8px`
- Radius: `cards: 8px, links: 8px, buttons: 9999px, navigation: 30px`

## Surface cues
- **Deep Forest Canvas** `#040c06`: Primary page and main section backgrounds.
- **Stone Slate Base** `#a49784`: Subtle background for persistent elements like navigation bars.
- **Parchment Highlight** `#eadfcf`: Accent backgrounds for cards, embedded forms, or informational panels, providing visual pop against the primary dark background.

## Component cues
- **Primary Action Button**: Calls to action that drive core user flow.
- **Parchment Card Button**: Secondary action or informational button within content areas.
- **Navigation Link**: Top-level navigation items.
- **Image Card**: Container for imagery, often with textual overlay.
- **Hero Headline**: Prominent, attention-grabbing titles for key sections.
