## Surface cues
- **System Canvas** `#e5e7eb`: Base page background
- **Pebble Gray** `#f3f4f6`: Slightly elevated backgrounds, subtle differentiation
- **Pure White** `#ffffff`: Primary surface for cards, panels, and content blocks
- **Ghostly Carbon** `#1f2937`: Darker surfaces for code blocks or specific card types

## Component cues
- **Primary Call-to-Action Button**: Main action button, drawing immediate attention.
- **Navigation Link Button**: Outlined button for navigation, less prominent than primary CTA.
- **Muted Code Card**: Card for displaying code snippets or technical details.
- **Testimonial Card**: Used for displaying customer testimonials or quotes.
- **Ghost Text Button (Interactive)**: Low-emphasis actions, often inline or secondary to primary actions.
