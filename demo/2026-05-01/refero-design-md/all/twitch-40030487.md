## Surface cues
- **Cloud Canvas** `#efeff1`: Base page background
- **Ghost White** `#ffffff`: Primary content areas, cards, interactive elements
- **Promotional Purple** `#330c6`: Accentuated banners or footer sections for brand emphasis

## Component cues
- **Primary Navigation Link**: Interactive element
- **Pill Button (Text Only)**: Secondary action button
- **Pill Button (Ghost White Text)**: Prominent action button on dark backgrounds
- **Pill Button (Muted Background)**: Subtle action button
- **Video Preview Card**: Content display
