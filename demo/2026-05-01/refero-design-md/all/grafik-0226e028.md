## Surface cues
- **Canvas Parchment** `#f0eeeb`: Dominant page background.
- **Interaction White** `#ffffff`: Backgrounds for embedded UI elements and certain content blocks over the main canvas.
- **Absolute Black** `#000000`: Backgrounds for header, footer, and striking full-width content sections, creating inversed visual contrast.

## Component cues
- **Navigation Link**: Primary navigation elements in the header and footer.
- **Content Block Frame**: Container for all visual and text content sections.
- **Project Metadata Text**: Descriptive text accompanying project visuals, often small and positioned within content blocks.
- **Time & Location Text**: Small, informational text like current time and location in the header.
