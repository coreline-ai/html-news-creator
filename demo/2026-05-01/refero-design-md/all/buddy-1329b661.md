## Surface cues
- **Cloud White Canvas** `#fcfcfd`: Primary page background, foundation for all content.
- **Frosted Card** `#fcfcfd`: Default card and container surfaces, subtle lift from canvas using border.
- **Elevated Accent Surface** `#00000000`: Cards and interactive elements that receive depth through border and tinted shadow.
- **Feedback Highlight** `#ffd888`: Distinct, highly visible accent surfaces for testimonials or important notices using gradients.

## Component cues
- **Primary Action Button**: Interactive element
- **Ghost Button**: Secondary action or navigation
- **Default Card**: Content container
- **Elevated Feature Card**: Prominent content container
- **Feedback Card**: Testimonial or distinct message
