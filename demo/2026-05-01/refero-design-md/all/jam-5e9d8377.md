## Typography direction
- **SF Pro Text**: 400, 500, 600, 700, 900, 11px, 12px, 13px, 14px, 15px, 16px, 24px, 1.00, 1.25, 1.33, 1.43, 1.50; substitute `Segoe UI, Arial, sans-serif`.
- **Oldschool Grotesk**: 400, 900, 8px, 12px, 16px, 66px, 88px, 116px, 1.00, 1.10, 1.50; substitute `Impact, sans-serif`.
- **SF Mono**: 400, 10px, 1.50; substitute `Courier New, monospace`.

## Spacing / shape
- Section Gap: `48px`
- Card Padding: `40px`
- Element Gap: `16px`
- Radius: `cards: 24px, pills: 9999px, buttons: 12px, default: 8px`

## Surface cues
- **Canvas** `#130f18`: Primary page background
- **Card Base** `#21192a`: Card and secondary container backgrounds
- **Elevated UI** `#ffffff`: Interactive UI panels and complex components (e.g., bug report modal)

## Component cues
- **Ghost Navigation Button (Text)**: Navigation and secondary actions
- **Feature Tag Button (Outlined)**: Informational tags and selectable filters
- **Primary Action Button**: Main call-to-action
- **Customer Testimonial Card**: Displaying customer feedback
- **Highlight Feature Card**: Highlighting key features or product aspects
