## Typography direction
- **sans-serif**: 400, 12px, 1.2.
- **Matter**: 400, 600, 12px, 14px, 16px, 18px, 20px, 24px, 32px, 40px, 48px, 56px, 64px, 0.90–1.56 (tight at display: 0.90–1.00; relaxed at body: 1.40–1.56); substitute `General Sans, Manrope, or DM Sans`.
- **Inter**: 400, 500, 14px, 16px, 1.00–1.38; substitute `Inter (already a Google font)`.

## Spacing / shape
- Section Gap: `80px`
- Card Padding: `16px`
- Element Gap: `10px`
- Page Max Width: `1200px`
- Radius: `cards: 16px, badges: 200px, images: 10px, inputs: 4px, buttons: 4px, largeCards: 20px, pillButtons: 50px`

## Surface cues
- **Deep Base** `#090909`: Absolute floor — nav overlay backgrounds, announcement bar
- **Canvas** `#121212`: Primary page background used across most sections
- **Elevated Card** `#1e1e1d`: Footer background, slightly raised content regions
- **Raised Surface** `#353534`: Card backgrounds, pill button fills, dropdown containers
- **Interactive Surface** `#40403f`: Hover states, focused overlay containers

## Component cues
- **Hero CTA Button Group with Code Snippet**: Reference component style.
- **Feature Section — Numbered List Card**: Reference component style.
- **Trust Logo Bar**: Reference component style.

- **Dark Filled Button**: Secondary actions within dark UI panels
