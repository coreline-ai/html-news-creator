## Surface cues
- **Pure Canvas** `#ffffff`: Base page background
- **Soft Sky Canvas** `#f0f8ff`: Subtle section backgrounds, creates gentle visual separation
- **Pure Canvas Card** `#ffffff`: Primary content cards, slightly elevated
- **Dark Code Card** `#373d42`: Elevated specific elements like code blocks or focused dark UI sections

## Component cues
- **Primary Action Button**: Filled button for main calls to action.
- **Ghost Button**: Minimal impact button for secondary actions or links.
- **Subtle Neutral Button**: Light background button for filters or toggles.
- **Clean Data Card**: Default card for displaying content blocks.
- **Elevated Data Card**: Emphasized card for key information or interactive components.
