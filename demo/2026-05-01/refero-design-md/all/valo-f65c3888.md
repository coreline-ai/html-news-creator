## Spacing / shape
- Section Gap: `45px`
- Card Padding: `18px`
- Element Gap: `20px`

- Radius: `links: 9999px, buttons: 9999px`

## Surface cues
- **Midnight Canvas** `#000000`: Primary page background and default surface for all content.
- **Silver Component Surface** `#ffffff`: Used sparingly for interactive elements or specific content blocks where a reversed, high-contrast display is desired within the dark theme.

## Component cues
- **Ghost Navigation Link**: Primary navigation links
- **Ghost Arrow Button**: Call to action for external links or advancing sections
- **Footer Link**: Secondary navigation and legal links
- **Subtle Divider Line**: Visual separator for content sections
