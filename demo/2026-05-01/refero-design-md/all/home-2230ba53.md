## Surface cues
- **Canvas Sand** `#e4dfd9`: Dominant page background
- **White Cloud** `#ffffff`: Elevated cards, main information containers
- **Ash Mist** `#c7c7c7`: Subtle background surfaces, decorative elements that are not interactive
- **Deep Plum** `#171717`: Interactive elements: button backgrounds, search bar backgrounds in hero

## Component cues
- **Ghost Navigation Button**: Header navigation, secondary actions
- **Talent/Employers Toggle**: Segmented control or primary category selection
- **Primary Action Button (Filled)**: Main calls to action
- **Search Input (Hero)**: Prominent search fields
- **Feature Card**: Content grouping, feature display
