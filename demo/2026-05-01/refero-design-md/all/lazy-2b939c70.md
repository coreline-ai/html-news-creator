## Surface cues
- **Base Canvas** `#26272c`: Main page background, primary canvas for content display.
- **Input / Elevated Base** `#1a1b1f`: Default background for interactive inputs and slightly elevated content blocks.
- **Interactive Card** `#3b3c3`: Background for interactive cards, receiving subtle shadow and border.
- **Overlay / Gradient Start** `#ffffff`: Used primarily as a starting point for gradients that lighten toward the top, or for highlight elements against very dark backgrounds.

## Component cues
- **Input Field**: Primary interactive input element
- **Elevated Tool Card**: Interactive content container with distinction
- **Primary Ghost Button**: Call to action or navigation link
- **Product Feature Card**: Neutral content display card
- **Pill Badge**: Categorization or short label
