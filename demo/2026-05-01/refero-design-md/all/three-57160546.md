## Component cues
- **Primary Action Button**: Filled button indicating a significant call to action.
- **Neutral Action Button**: Filled button for secondary actions or navigation.
- **Ghost Button**: Low-prominence button for less critical actions.
- **Feature Card**: Container for showcasing features or testimonials.
- **Text Input (White)**: Standard input field for user entry.
