## Typography direction
- **FKGrotesk**: 400, 500, 700, 16px, 20px, 24px, 44px, 64px, 80px, 1.00, 1.09, 1.25, 1.40, 1.50; substitute `Inter`.
- **Mona Sans**: 600, 12px, 1.17; substitute `Roboto Mono`.

## Spacing / shape
- Section Gap: `24px`
- Card Padding: `16px`
- Element Gap: `8px`
- Radius: `inputs: 4px, buttons: 4px, default: 4px`

## Component cues
- **Primary Action Button**: Highest emphasis button, used for main calls to action.
- **Ghost Button**: Low emphasis, secondary action button.
- **Text Input Field**: Standard input for user data.
- **Dropdown Navigation Item**: Navigation links with hover/active states.
- **Info Card**: Container for grouped information, often used in grids.
