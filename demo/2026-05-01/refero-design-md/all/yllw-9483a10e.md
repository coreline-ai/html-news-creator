## Surface cues
- **Canvas White** `#ffffff`: Primary page background, expansive light areas.
- **Section Fog** `#cac7b4`: Secondary page sections, alternating background for content blocks.
- **Utility Yellow** `#ffdd00`: Top-level alert banners, highly prominent calls to action or information.
- **Accent Dark** `#191919`: High-contrast background for feature blocks or quotes, providing visual anchor.

## Component cues
- **Filled Button - Foggy**: Primary action button for key calls to action.
- **Outlined Button - Foggy**: Secondary action button, providing a less prominent interactive element.
- **Filled Button - White**: Action button used on darker backgrounds or for specific content blocks.
- **Outlined Button - White Text**: Ghost button variant, typically used on dark or image backgrounds.
- **Borderless Card**: Content container for features or descriptions, emphasizing content without visual heavy lifting.
