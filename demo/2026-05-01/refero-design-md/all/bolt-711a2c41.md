## Surface cues
- **Page Canvas Light** `#f8f6fe`: Base background for lighter page sections.
- **Page Canvas Dark** `#04091a`: Dominant background for main content sections and footer, establishing a deep, secure tone.
- **Elevated Surface** `#111426`: Used for cards, input fields, and badges, providing subtle visual separation on dark backgrounds.

## Component cues
- **Primary Action Button**: Main call-to-action
- **Ghost Navigation Button**: Secondary navigation action
- **Subtle Link Button**: Informational/utility button
- **Navigation Link**: Header navigation item
- **Input Field**: User data entry
