## 개발 목적

현재 AI 데일리 리포트가 “AI 관련 자료 모음”처럼 보이는 문제를 해결한다. 핵심은 자동 파이프라인에 **편집 정책(editorial policy)** 과 **출처 대중성/중요도 랭킹**을 넣어, 하루 단위로 대중적이고 중요한 AI 뉴스를 우선 선별하는 것이다.

## 개발 범위

- AI 데일리 리포트용 편집 정책 YAML 추가
- source tier/카테고리/우선순위 metadata 정리
- editorial ranker 추가
- arXiv/GitHub/공식 블로그/대중 매체 간 우선순위 차등화
- `run_daily.py` 생성 단계에서 섹션 후보를 editorial score 순으로 제한
- 품질 게이트: 출처 없는 섹션 금지, arXiv-only 메인 후보 제한, fallback 이미지 유지
- 병렬 에이전트 작업 병합과 테스트 검증

## 제외 범위

- 관리자 대시보드
