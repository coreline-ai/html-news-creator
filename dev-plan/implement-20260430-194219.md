이 문서는 이번 개발의 범위를 고정하고, 구현이 목적 밖으로 확장되지 않도록 하기 위한 작업 문서다.

## 개발 목적
- 기존 `light` / `dark` 테마를 유지하면서, 외부 공개·스크린샷·뉴스룸형 공유에 적합한 3번째 `newsroom-white` 테마를 추가한다.
- 단순 흰 배경이 아니라 출처, 대표 이미지, 통계/요약 박스가 더 명확하게 읽히는 공개용 뉴스 리포트 테마를 구현한다.

## 개발 범위
- `templates/report_newsstream.html.j2`에 `data-theme="newsroom-white"` CSS 토큰과 컴포넌트 보정 스타일을 추가한다.
- 테마 토글을 `light → dark → newsroom-white → light` 순환 방식으로 변경한다.
- 접근성용 `aria-label` / `title` 문구를 현재 테마 상태와 다음 전환 대상으로 맞춘다.
- 신규 테마 상태가 시각적으로 확인되도록 토글 버튼에 3번째 아이콘을 추가한다.
- 기존 `light` / `dark` 테마의 색상·형태는 변경하지 않는다.

## 제외 범위
- `base.html.j2`, `section_card.html.j2` 전체 리디자인은 제외한다.
- 리포트 레이아웃 구조 변경, 섹션 구성 변경, 파이프라인 로직 변경은 제외한다.
- 뉴스 이미지 선정/수집 로직 변경은 제외한다.
- 이미 생성된 `public/news/*.html` 직접 수정은 제외한다. 필요한 경우 렌더러로 재생성한다.
